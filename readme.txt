Projeto Airbnb Rio - Ferramenta de Previsao de Preço de Imóvel

Contexto:
Projeto feito ao estudar o curso de python da hashtag 

Objetivo do Projeto:
Estudar o conteudo de manipulação de dados e ciencia de dados. Para isso, utilizei uma base de 
dados com o intuito de treinar uma inteligencia artificial para prever preços de imoveis de 
padrão médio do Rio de Janeiro para alugar no airbnb. 

dataset: https://www.kaggle.com/datasets/allanbruno/airbnb-rio-de-janeiro/data
