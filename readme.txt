Projeto Airbnb Rio - Ferramenta de Previsão de Preço de Imóvel

Contexto:
Projeto feito ao estudar o curso de python da hashtag 

Objetivo do Projeto:
Estudar o conteúdo de manipulação de dados e ciência de dados. Para isso, utilizei uma base de 
dados com o intuito de treinar uma inteligência artificial para prever preços de imóveis de 
padrão médio do Rio de Janeiro para alugar no airbnb. 

dataset: https://www.kaggle.com/datasets/allanbruno/airbnb-rio-de-janeiro/data
